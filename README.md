# Tesla-Coil-Business-Cards

Your standard PCB business cards. Except, these have an entire Tesla coil built inside of them!

The popular slayer exciter circuit is used here. Since it's self tuning, there is no manual tuning for the egregious resonant frequency that I know this coil has.
It's between 2-3MHz, so the 2N2222 transistor should be able to handle it accordingly.
I doubt there will be plasma arcs, however, neon bulbs may be able to light up.

This project was made on EasyEDA and qualifies for the OSHWLAB Stars sponsorship.
